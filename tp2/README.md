## Rappel: scanf

Un programme peut lire un entier en entrée comme ceci:

```c
int mon_entier;
scanf("%d", &mon_entier);
```

et une chaîne de caractères comme cela:

```c
char ma_chaine[30];
scanf("%s", ma_chaine);
```
